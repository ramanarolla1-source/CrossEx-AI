# CrossEx AI – Demo Resources

## From Witness Statements to Cross-Examination Questions in Minutes

CrossEx AI is an AI-native litigation workflow designed to help lawyers transform witness statements into structured cross-examination preparation.

This page contains all primary project resources for reviewers, judges, and contributors.

---

# Project Overview

CrossEx AI transforms witness statements into litigation-ready questioning strategies through a structured workflow:

```text
Witness Statement
        ↓
Fact Extraction
        ↓
Timeline Analysis
        ↓
Contradiction Detection
        ↓
Question Generation
        ↓
Lawyer Review
        ↓
Export
```

The application is built as an Anna-native workflow that combines AI-assisted analysis with mandatory human review.

# Example Use Case

### Witness Statement

"I arrived at the intersection at 8 PM."

Later:

"I reached the location around 7:45 PM."

### CrossEx AI Analysis

Detected Issue:

Timeline inconsistency.

Generated Question:

"You previously stated that you arrived at the intersection at 8 PM. Why did you later state that you reached the location around 7:45 PM?"
